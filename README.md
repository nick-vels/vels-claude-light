# Vels Claude Light

**Личная AI-Агентная система в Telegram — на базе Claude Code**

*Пишите задачи боту как обычному коллеге — он сам напишет, поправит и запустит код.*

</div>

---

## 🤔 Что это такое?

Vels Claude Light — это программа-«мост» между **Telegram** и **Claude Code** 

Проще говоря: вы пишете сообщение в Telegram — а на сервере за вас работает ИИ-программист, который читает ваш код, правит его, создаёт файлы и отчитывается вам обратно в тот же чат.

## 💡 Зачем это нужно?

Обычно, чтобы пользоваться Claude Code, нужно сидеть за компьютером с открытым терминалом. Это неудобно: с телефона не зайдёшь, с планшета тоже.

С этим ботом:

- 📱 **Работаете с телефона.** Идёте в метро — отправили задачу, вернулись домой — код уже готов.
- 🌙 **Бот работает 24/7.** Поставили задачу на ночь, утром читаете результат.
- 🖼 **Можно слать картинки и файлы.** Скриншот ошибки, макет, логи — бот всё поймёт.
- 🔒 **Только вы имеете доступ.** Бот отвечает только вам (по вашему Telegram ID).

Этот проект создан специально для участников **Практикума**, чтобы каждый мог завести себе личного ИИ-ассистента без головной боли.

## ✨ Что умеет бот?

- Отвечать текстом на любые вопросы про ваш код
- Читать, создавать и править файлы в вашей рабочей папке
- Принимать картинки (скриншоты ошибок, схемы, макеты)
- Принимать файлы (логи, документы, исходники)
- Стримить ответ по мере генерации — не ждёте молча по минуте, а видите, как ИИ думает
- Помнить контекст разговора (как обычный чат)
- Выполнять несколько задач по очереди, если пишете быстрее, чем он отвечает

## 🚀 Установка — по шагам

> **Не пугайтесь количества шагов** — реальной работы здесь минут на 15, большую часть делает установщик.

### Что вам понадобится

1. **VPS** (виртуальный сервер) с Ubuntu 22.04+ или Debian 12+.
   *Где взять:* Timeweb, Beget, Aeza, Hetzner — любой за 200–400₽/мес. Нужен самый маленький тариф.
2. **Аккаунт в Telegram** — у вас уже есть.
3. **Аккаунт Anthropic** (для Claude Code) — регистрация на [claude.ai](https://claude.ai). Платная подписка обязательна: Claude Code входит в Pro/Max-тариф.
4. **15 минут свободного времени.**

---

### Шаг 1. Создайте Telegram-бота

1. Откройте в Telegram [@BotFather](https://t.me/BotFather)
2. Отправьте ему команду `/newbot`
3. Придумайте имя бота (например, *Мой Claude*)
4. Придумайте username бота — он должен заканчиваться на `_bot` (например, `my_claude_assistant_bot`)
5. BotFather пришлёт вам **токен** — длинную строку вида `1234567890:AAF...XyZ`

   📌 **Сохраните этот токен — он понадобится при установке.**

### Шаг 2. Узнайте свой Telegram ID

1. Откройте в Telegram [@userinfobot](https://t.me/userinfobot)
2. Нажмите `Start`
3. Бот пришлёт ваш ID — это число (например, `123456789`)

   📌 **Сохраните этот ID — он нужен, чтобы никто посторонний не писал вашему боту.**

### Шаг 3. Подключитесь к VPS

После покупки VPS провайдер пришлёт вам:
- IP-адрес сервера (например, `85.120.84.19`)
- Логин (обычно `root`)
- Пароль

Подключитесь к серверу:

**С Mac/Linux:**
```bash
ssh root@85.120.84.19
```
(замените IP на свой)

**С Windows:** скачайте бесплатный [MobaXterm](https://mobaxterm.mobatek.net/) или используйте встроенный PowerShell с той же командой.

При первом подключении введите `yes` и пароль, который прислал провайдер.

ВАЖНО! На сервере необходимо создать нового пользователя:

```bash
sudo adduser <username>
```

(Опционально) Дать sudo                                                                                     
```bash
sudo usermod -aG sudo <username>
```

### Шаг 4. Установите Claude Code CLI

Прямо в открытом SSH-соединении выполните:

```bash
apt update && apt install -y nodejs npm
npm install -g @anthropic-ai/claude-code
claude
```

Последняя команда `claude` запустит авторизацию:
- откроется ссылка — скопируйте её в браузер
- войдите в свой Anthropic-аккаунт
- вставьте обратно в терминал код подтверждения

Когда увидите приглашение вида `>` — нажмите `Ctrl+C` и выйдите из `claude`. Авторизация сохранена.

### Шаг 5. Запустите установщик бота

Одна команда — и установщик всё сделает сам:

```bash
curl -sSL https://raw.githubusercontent.com/nick-vels/vels-claude-light/main/install.sh | sudo bash
```

Установщик задаст **три вопроса**:

1. **Токен Telegram-бота** — вставьте тот, что получили от BotFather (Шаг 1)
2. **Ваш Telegram ID** — вставьте тот, что получили от userinfobot (Шаг 2)
3. **Рабочая директория** — нажмите Enter, оставив значение по умолчанию

Установщик:
- проверит, что токен рабочий
- скачает код
- создаст системного пользователя `vels-bot` (чтобы бот работал от отдельного аккаунта — это безопаснее)
- установит Python-зависимости
- запустит бота как системный сервис (будет перезапускаться сам при любых сбоях и после перезагрузки сервера)

Когда увидите сообщение **«✅ Установка завершена»** — готово.

### Шаг 6. Проверьте, что бот работает

1. Откройте Telegram
2. Найдите своего бота по имени пользователя (которое задали в Шаге 1)
3. Напишите ему: `Привет!`
4. Бот должен ответить — он передаст ваше сообщение в Claude

🎉 **Готово! Ваш личный ИИ-программист к вашим услугам.**

## 💬 Как пользоваться

### Обычные сообщения

Пишите всё что угодно — обычным текстом, как человеку:

> *Напиши скрипт на Python, который скачивает курс доллара с сайта ЦБ*

> *В файле `app.py` есть ошибка, помоги разобраться — вот логи (прикреплён файл)*

> *Вот скриншот дизайна (прикреплена картинка), свёрстай такую же страницу на HTML*

### Команды

| Команда | Что делает |
|---|---|
| `/new` или `/clear` | Начать новый разговор — бот забудет предыдущий контекст |
| `/compact` | Попросить Claude сжать историю разговора (если чат стал длинным) |
| `/stop` | Прервать ответ, если Claude пишет слишком долго или не туда |
| `/status` | Показать информацию о текущей сессии |

### Несколько советов

- **Формулируйте чётко.** «Исправь ошибку в коде» — плохо. «В файле `main.py`, строка 42, функция `get_user()` падает с `KeyError` — покажу логи ниже» — хорошо.
- **Прикрепляйте контекст.** Скриншот, лог-файл, ссылку на проблему — чем больше деталей, тем точнее ответ.
- **Используйте `/new`** когда переходите к новой задаче. Иначе бот продолжит обсуждать старое.
- **Файлы Claude кладёт в рабочую папку** (по умолчанию `/var/lib/vels-bot/workspace/`). Если нужно забрать файл с сервера — скажите боту «пришли мне файл X», он пришлёт его вложением в Telegram.

## 🛠 Управление ботом

Все команды выполняются в SSH-соединении (Шаг 3).

```bash
sudo systemctl status vels-claude-light    # статус (работает или нет)
sudo journalctl -u vels-claude-light -f    # живые логи (Ctrl+C для выхода)
sudo systemctl restart vels-claude-light   # перезапустить
sudo systemctl stop vels-claude-light      # остановить
sudo systemctl start vels-claude-light     # запустить снова
```

---

## 🗑 Как удалить

Одна команда:

```bash
sudo /opt/vels-claude-light/uninstall.sh
```

Скрипт спросит:
- удалить ли рабочую папку (по умолчанию — нет, оставит)
- удалить ли пользователя `vels-bot` (по умолчанию — да)

После этого от бота на сервере не останется ничего.

## ❓ Если что-то пошло не так

### Бот не отвечает в Telegram

```bash
sudo systemctl status vels-claude-light
```

Если написано `failed` или `inactive` — смотрите логи:

```bash
sudo journalctl -u vels-claude-light -n 50
```

В логах обычно видно причину: неправильный токен, нет доступа к Claude, проблема с сетью.

### «claude CLI не найден» при установке

Значит Шаг 4 не был выполнен. Вернитесь и установите Claude Code CLI.

### «Telegram отказал» при установке

Токен неверный. Проверьте, что скопировали его целиком из BotFather.

### Бот отвечает «доступ запрещён»

Вы пишете боту не с того аккаунта. Проверьте, что ваш ID в [@userinfobot](https://t.me/userinfobot) совпадает с тем, что указали при установке. Если хотите добавить ещё один ID — откройте `/opt/vels-claude-light/.env`, найдите строку `ALLOWED_USER_IDS=` и допишите через запятую, потом `sudo systemctl restart vels-claude-light`.

### Claude отвечает слишком долго / зависает

Отправьте `/stop`, потом `/new` и попробуйте заново. Если проблема повторяется — проверьте, что у Claude Code на этом сервере работает подписка (`claude` из SSH должен запускаться без ошибок).

## 📚 Для любопытных: что внутри

- `src/bot.py` — обработчик сообщений Telegram (библиотека [aiogram](https://aiogram.dev/))
- `src/bridge.py` — запускает `claude -p` как подпроцесс и собирает ответ
- `src/streaming.py` — показывает ответ по мере генерации (редактирует одно сообщение)
- `src/uploads.py` — сохраняет картинки и файлы из Telegram, чтобы Claude их видел
- `src/storage.py` — помнит ID сессий Claude для каждого чата
- `scripts/vels-claude-light.service` — настройка systemd-сервиса
- `install.sh` / `uninstall.sh` — установщик и деинсталлятор

## 💻 Для разработчиков: запуск локально

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env    # отредактируйте .env: токен, ID, путь
python -m src.main
```
